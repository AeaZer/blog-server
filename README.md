## 上手指南
### 服务端启动
1. 克隆项目到本地 
1. 找到项目中resources目录下的myblog.sql文件，在MySQL数据库中执行(需要Mysql版本5.7及以上).
1. 启动Redis服务(Redis和Mysql配置在项目的application.properties中)
1. 在IntelliJ IDEA中运行本项目
1. 至此，服务端就启动成功了，此时我们直接在地址栏输入 http://localhost 即可访问我们的项目
